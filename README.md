# GPA-Calculator

## 📊 Overview
A modern, full-featured web application for tracking academic performance across multiple semesters with GPA calculation, data persistence, and professional PDF reporting.

## ✨ Key Features

### 🔐 User Authentication
- Secure registration/login system
- Individual user accounts with personalized data storage
- Password-protected academic records

### 📚 Semester Management
- Create and manage multiple academic semesters
- Add/edit/delete courses with custom names, credits, and grades
- Real-time GPA calculation with visual feedback
- Edit previously saved semesters

### 📈 GPA Tracking
- Individual semester GPA calculation
- Cumulative GPA across all semesters
- Interactive GPA progression chart (Chart.js)
- Performance classification (Excellent, Good, Satisfactory, etc.)
- Credit-weighted calculations

### 📄 PDF Export System
- Export individual semester reports
- Generate complete academic history PDF
- Progress reports with performance analysis
- Professional formatting with tables and summaries
- Print-ready documents

### 🎨 Modern UI
- Clean, responsive design
- Interactive semester cards
- Real-time notifications
- Mobile-friendly layout
- Smooth animations

## 🛠️ Tech Stack
- **HTML5** - Semantic structure
- **CSS3** - Flexbox/Grid, custom properties
- **JavaScript (ES6+)** - Core logic
- **Chart.js** - Data visualization
- **jsPDF** - PDF generation
- **LocalStorage** - Client-side persistence
- **Font Awesome** - Icons
